Para rodar todo o sistema utilizamos o comando:
npm run dev
-----------------------------------------------

[App.jsx] é onde iniciamos o nosso sistema, arquivo principal
[main.jsx] é  o ponto de entrada da aplicação, estamos importando o React e o ReactDOM para renderizar o componente principal App dentro do elemento HTML com id 'root'. 

index.html > executa o main.jsx, e o main.jsx > executa o App.jsx
